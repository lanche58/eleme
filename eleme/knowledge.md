# 项目中的学到的知识

## linkActiveClass和linkExactActiveClass

#### linkActiveClass

类型: string

默认值: "router-link-active"

全局配置 <router-link> 的默认『激活 class 类名』。

#### linkExactActiveClass

2.5.0+

类型: string

默认值: "router-link-exact-active"

全局配置 <router-link> 精确激活的默认的 class。
